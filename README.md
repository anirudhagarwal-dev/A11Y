# Accessible Portfolio

Simple, accessible personal portfolio. Open `index.html` directly in a browser or use a local server.
---

## Features
- Skip link, section-aware navigation (`aria-current`)
- Accessible form validation with `aria-invalid` and alert region
- Dark mode and high-contrast support
- Smooth scrolling respecting reduced motion
---

## Files
- `index.html` — main page
- `style.css` — styles
- `favicon.png`, `mypic.jpg` — assets
